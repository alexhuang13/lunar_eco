# 🌙 天外家园 · Version 6 更新说明

本版围绕**人物深度**展开:用「天赋系统」取代旧职业,新增「医疗 / 受伤系统」与「药物生产链」,并把建造改为需要投入工时的「建造工作」。

---

## 1. 环保:删除月球排废
- 移除「排向月壤的废水 / 固体废物」选项,不再允许向月球环境倾倒污染物。

## 2. 作物名简化
- 去掉作物英文型号,只保留中文名:**矮秆小麦 🌾 / 气雾生菜 🥬 / 微型土豆 🥔**。
- 新增 **药用植物 🌿**(周期 70 天,收获产「药材」而非食物)。

## 3. 天赋系统(取代职业)
- 每名角色拥有 **0~2 个天赋**(可双天赋),随机生成。数量概率:0 个 30% / 1 个 50% / 2 个 20%。
- 天赋类型:**科研 / 探索 / 修理 / 建造**(工作类,放大对应工作产出)与 **娱乐**(每天给全员加心情)。
- **工作类倍率**:顶级 ×3.0 / 高级 ×2.0–2.9 / 中级 ×1.5–1.9 / 低级 ×1.0–1.4。
- **娱乐加成**(每人每天心情%):顶级 +2.0 / 高级 +1.7–1.9 / 中级 +1.3–1.5 / 低级 +1.0–1.2。
- 随机天赋等级分布:低级 45% / 中级 35% / 高级 20%(**顶级仅四部长拥有**)。
- **姓名随机**(普通人名,中英混合,避开职业名与政治人物)。
- **开局选一位部长**作为 1 号成员,其余携带人员姓名+天赋全随机:
  - Annie 部长 — 顶级建造(建造速度 ×3)
  - 黄部长 — 顶级科研(科研速度 ×3)
  - 罗部长 — 顶级探索(探索收获 ×3)
  - 郭部长 — 顶级娱乐(每天全员 +2% 心情)

## 4. 建造工作(新增日常工作)
- 扩建改为**排队建造**:下达扩建后进入建造队列,需指派乘员做「🏗️ 建造」工作逐步建成。
- 开工时一次性扣除构造水;**建造工作量(人·天)**:居住舱 3 / 种植舱 4 / 堆肥舱 2 / 实验室 5 / 温室 6。
- 建造天赋者推进更快(Annie 部长 ×3)。

## 5. 招募成员
- 运营中满足条件(居住容量有余 + 食物/水为正)可招募新成员,姓名+天赋随机生成。

## 6. 医疗系统(疾病)
- 每人初始**体质 0.7–1.0**(不会特别差),体质低更易生病、更难自愈。
- **发病**:每日基础 3%,受体质与心情调节(好心情降发病、坏心情升发病)。
- **病程**:初期有概率自愈(体质好概率大);连续 3 天未愈转「加重」,需**吃药**(每天 1 单位药物,60% 好转)。无药则每天扣健康。
- 好心情减少生病概率、增大康复概率、缩短休养;坏心情相反。

## 7. 受伤系统
- 仅**外出探索**时触发(每人每日 8%),分轻伤 / 重伤。
- 轻伤有概率自愈;重伤需吃药恢复,之后仍需恢复期休养。好心情提高自愈概率、缩短修养。

## 8. 药物生产链
- **药用植物**(种植园)收获 → 产「**药材**」。
- **制药课题**(实验室新增,算科研任务,保留原有课题):消耗药材产出「**药物**」,可重复进行。
- 药物用于治疗疾病与重伤。

## 9. 伤病与工作
- 伤病状态在**人物状态栏**标明,并实时影响健康值。
- 带伤病可工作,但**效率随健康度下降**,且健康会随工作时间下降。

## 10. 永久移除机制
- **健康降到 0 → 死亡**(永久移除)。
- **心情降到 0 → 叛逃**(永久移除)。

## 11. 界面
- 中文字体修复(云端 Linux 正常显示)。
- **时间流逝(天数)移至左栏最上方**。
- 资源面板新增药材 / 药物;人物状态显示天赋(及等级)与伤病状态。
