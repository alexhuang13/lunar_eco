> 本文档**只做设计与数据论证，暂不修改代码**。包含两项需求：（1）新增净水量指标；（2）在基础舱内种植以改造为温室舱。

## 需求一：净水量指标（Water Reclamation Rate）

### 现状问题

当前代码里只有一个**净水总量** `Clean_Water_kg`（蓄水量，存量），但没有单独的「净水生产速率」（流量）指标。净水其实由多条途径产生，散落在各处、没有汇总：

- 微藻净化废水：`alg_pur`（第 6 段）
- 作物蒸腾回流：`base_evap` + 各批次 `act_cw`（第 7 段）
- 焚化炉副产水：进的是废水池，不算净水

玩家无法直观看到「每天到底净化了多少水、够不够用」，只能看总量涨跌反推。

### 设计：新增「每日净水产量」流量指标

把每日所有净水来源**汇总为一个可监控的流量值** `daily_water_reclaimed`，与「每日净水消耗」并列展示，让玩家一眼看出水循环是否收支平衡。

```
每日净水产量 = 微藻净化量 + 作物蒸腾回收量 + 温室舱蒸腾回收量(见需求二)
每日净水消耗 = 乘组生活用水 + 建造注水 + 舱室维护损耗
净水净流量   = 产量 − 消耗   （正=盈余，负=亏空）
```

仪表盘建议新增：
- 一个 metric 卡片「今日净水产量 X kg / 消耗 Y kg」，颜色随净流量正负变化；
- 历史曲线里增加「每日净水产量」一条线，与现有 `Clean_Water` 存量曲线并列，区分**流量**与**存量**。

### 真实数据依据

净水回收是闭环生命支持的核心。ISS 俄罗斯舱段的 SRV-K 系统**从 2000 到 2020 年间，由湿气冷凝水回收了 16155 升水，约占乘组饮用水需求的 63%、总需水量的 37%**。地面研究中，膜生物反应器 + 反渗透的混合系统可对湿气冷凝水实现**约 90% 的回收率**。

- ISS 冷凝水回收占比（63% 饮用水）：https://ttu-ir.tdl.org/server/api/core/bitstreams/ac8532e0-75f2-4268-81ab-c1bc2f5f817a/content
- 混合系统 90% 回收率（湿气冷凝水→近饮用水）：https://www.ncbi.nlm.nih.gov/pmc/articles/PMC12298372/

> 这说明「冷凝/蒸腾回收」在现实中就是主力水源，把它做成显式指标完全符合真实工程逻辑。

## 需求二：基础舱内种植 → 改造为温室舱（Greenhouse Conversion）

### 现状逻辑

当前有三类独立舱室：居住舱（`hab`）、种植舱（`plant`）、堆肥舱（`compost`），数量各自独立增减。种植是在专用「种植舱」里进行的。

### 新逻辑：温室舱是「居住舱 + 种植」的复合升级

允许在**基础（居住）舱内种植植物**，将其改造为**温室舱**。温室舱是一种兼具居住与种植功能的复合舱：绿色空间环绕生活区，既产出资源又极大改善心理状态（呼应现实中绿植对密闭环境乘组的心理疗愈作用）。

**温室舱的收支特征（相对普通居住舱）：**

| 项目 | 方向 | 说明 |
|------|------|------|
| 肥料消耗 | ⬆ 更多 | 舱内植物需持续施肥 |
| 废水消耗 | ⬆ 更多 | 灌溉吸收废水 |
| 二氧化碳消耗 | ⬆ 更多 | 光合固碳 |
| 电量消耗 | ⬆ 更多 | LED 补光是大头（呼应 v2 电力系统） |
| 氧气产出 | ⬆ 少量 | 光合产氧 |
| **净水产出** | ⬆ 少量 | **植物蒸腾→冷凝回收**（接入需求一的净水指标）|
| **心情度** | ⬆⬆ 大幅提升 | 绿色生活空间的心理疗愈 |

### 真实数据依据

EDEN ISS（DLR 在南极的太空温室）的运行经验表明：温室是**半闭环**系统，植物在吸收光与热的同时大量蒸腾水分，回收的蒸腾水需循环再利用以节约系统内有限的水；其专门设有「冷凝水回收子系统」来处理这部分蒸腾水。理想生长温度 20–25 ℃，热量主要来自 LED 光源——印证了温室「高耗电（补光）+ 产净水（蒸腾冷凝）」的双重特征。

- EDEN ISS 温室 FAQ（蒸腾、湿度控制、水回收）：https://eden-iss.net/wp-content/uploads/4_EDENISS_FAQ.pdf
- EDEN ISS 冷凝水回收子系统（DLR）：https://elib.dlr.de/186911/
- 太空植物栽培白皮书（蒸腾维持湿度平衡）：https://www.sciencedirect.com/science/article/pii/S221455242400066X
- EDEN ISS 植物健康监测（南极实验阶段）：https://www.frontiersin.org/journals/plant-science/articles/10.3389/fpls.2019.01457/full

> 关键现实依据：温室的高耗电（补光）与产净水（蒸腾冷凝）是同一套物理过程的两面，本设计如实还原了这一点。

### 建议的数值方向（待你确认后细化）

以单个温室舱 / 天为单位，相对普通居住舱的增量（具体系数实装时再调平衡）：

| 参数 | 建议量级 | 关联现有变量 |
|------|---------|-------------|
| 额外肥料消耗 | ~1–2 kg | `Fertilizer_kg` |
| 额外废水消耗 | ~10–15 kg | `Waste_Water_kg` |
| 额外 CO₂ 消耗 | ~0.5–1 kg | `CO2_kg` |
| 额外电量消耗 | ~20–30 kWh | （v2 电力系统） |
| 氧气产出 | ~0.3–0.5 kg | `O2_kg` |
| 净水产出 | ~5–10 kg | `Clean_Water_kg` + 净水指标 |
| 心情加成 | 显著高于普通种植舱的 `plant*0.6` 项 | 各成员 `mood` |

## 系统逻辑需要修改的地方（实装清单，供确认）

1. **舱室模型**：在 `hab` / `plant` / `compost` 之外，新增温室舱类型 `greenhouse`（或将居住舱标记「已绿化」状态），需决定它在体积、注水、维护上的归类。
2. **每日循环新增温室结算段**：在现有第 6（微藻）、第 7（作物）段之间或之后，加入温室舱的「耗肥/耗水/耗碳/耗电 → 产氧/产净水/加心情」结算。
3. **净水指标**：在每日循环里累加 `daily_water_reclaimed`，写入 history，并在仪表盘新增 metric + 曲线。
4. **心情公式**：在现有 `delta_m` 里，为温室舱加一个明显高于普通种植舱的正向项。
5. **电力耦合**：温室耗电接入 v2 电力系统（若电力先行）；若电力尚未实装，可先用占位常量。
