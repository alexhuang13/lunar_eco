# 天外家园：无限扩展生态沙盒 —— 扩展模块说明

本版本在原始闭环生态沙盒的基础上，新增了 **突发事件系统**，并规划了 **心情值系统的精细化重构**。下文记录扩展部分的设计思路、参数映射与调参入口。

---

## 一、突发事件系统（已实现）

### 设计原则

突发事件不是凭空扣血，而是 **精确打击对应的 state 参数，并沿系统耦合关系级联**。例如撞击穿孔会同时损失大气（O₂/CO₂）和削减壳体完整性（σ），而不是单纯扣一管氧气。

事件按 **起病速度 / 预警窗口 / 应对模式** 分为三类，对应三种时间尺度：

| 类别 | 时间尺度 | 应对模式 | 机制 |
|------|----------|----------|------|
| `instant` | 秒 – 分钟 | 自动反射 | 一次性瞬时冲击参数，靠事前裕度抵御 |
| `ongoing` | 分钟 – 小时 | 人工应急 | 持续若干天的减益状态，留有响应窗口 |
| `creeping` | 天 – 周 / 月 | 趋势预警 | 慢变量缓慢漂移，初期几乎无征兆 |

### 事件清单与参数映射

| 事件 | 类别 | 冲击参数 | 说明 |
|------|------|----------|------|
| ☄️ 微流星体撞击穿孔 | instant | `O2_kg` · `CO2_kg` · `hull_integrity` | 失压跑气 + 结构受损（σ + A 耦合） |
| 🌟 太阳高能粒子事件 (SPE) | instant | `mood` · `hull_integrity` | 伤害由月壤护甲减免，护甲足够厚可完全免疫 |
| ⚡ 主电源中断 | instant | `light` · CDRA | 当日光合停摆 + 气压调节失效 |
| 🌡️ 热控回路泄漏 | ongoing (4天) | `mood` | 舱内升温，持续热应激 |
| 🔧 ECLSS 净化部件故障 | ongoing (5天) | CDRA | CO₂ 洗涤失效，毒性易堆积 |
| 🌪️ 火星/月面尘暴 | ongoing (7天) | `light` | 遮蔽阳光，光照效率大降 |
| 🫧 密封件缓慢泄漏 | creeping (25天) | `O2_kg` · `CO2_kg` | 每天微量失气，长期累积致命 |
| 🦠 杂菌生物膜暴发 | creeping (15天) | `Clean_Water` · `Waste_Water` · `Algae_Biomass` | 污染净水 + 与微藻争养分 |
| 🧱 结构辐照脆化 | creeping (30天) | `hull_integrity` | 壳体缓慢脆化，临界前无征兆 |

### 关键耦合点

- **`hull_integrity`（壳体结构完整性 σ）** 是新增的核心枢纽参数：撞击、辐射、脆化均叠加于此，归零即触发「舱体解体失压」死亡判定，补上了原版只有大气崩溃、缺结构崩溃的空白。
- **月壤护甲（`Regolith_Shield_m2`）成为实质抗灾能力**：SPE 伤害随护甲厚度递减，约 250 m² 以上基本免疫。此前的护甲建设投入因此真正变得有价值。
- **断电不直接扣资源，而是关闭光合与 CDRA**，通过既有逻辑自然级联出缺氧/CO₂ 失控的后果。

### 控件与展示

- 侧边栏「⚠️ 突发事件强度」滑块：关闭 / 低 / 中 / 高，对应每日触发概率 0% / 4% / 10% / 20%。
- 仪表盘新增：壳体完整性进度条、进行中事件红色横幅、突发事件日志、图表中的壳体完整性曲线。

### 调参入口

- 事件触发概率：侧边栏 `event_chance` 字典。
- 慢变量衰减强度：`EVENT_LIBRARY` 中 `slow_leak`（0.992）、`embrittlement`（−0.8/天）等系数，初始值偏温和，可按需调狠。
- 各事件权重：`EVENT_LIBRARY` 中每个事件的 `weight` 字段，控制相对出现频率。

## 二、心情值系统精细化重构（规划中）

将当前的逐人心情值进一步完善，使其成为反映乘组状态的综合指标。

### 目标规则

1. **逐人独立 + 命名**：每个成员独立维护 `mood`（0–100）与 `name`，仪表盘展示逐人心情明细表（而非仅平均值）。
2. **基础自然衰减（随机但温和）**：每人每天因长期驻留而心情下降，幅度逐人随机；但设定上乘组经过严格筛选、心理素质过硬，衰减须缓慢（建议每人每天 0.1–0.5 随机区间）。
3. **三阶段分级警报**：
   - 高位（>70）：状态良好，无警报；
   - 中位（40–70）：黄色警告，幽闭焦虑；
   - 低位（<40）：红色警报，崩溃/叛乱边缘。
   - 按全员平均判定整体状态，任一成员跌入低位则单独高亮其姓名。
4. **温室舱提升心情**：种植舱越多，绿色空间的心理疗愈加成越大。
5. **总人数群体效应**：基地总人数越多，孤独感越低，衰减越缓慢（社会支持）。
6. **单房间拥挤惩罚**：单个居住舱平均人数越多越拥挤，衰减越快。规则 5 与 6 是一对相反的力，需分别计算后叠加。
7. **资源状态影响**：食物、O₂、CO₂、废水/固废越接近危险阈值，对心情的负面影响越大（整合现有 `resource_stress` 逻辑，避免重复计算）。

### 综合公式（建议形式）

```
每日心情变化量 = 基础随机衰减(−)
              + 温室加成(+)
              + 总人数加成(+)
              − 拥挤惩罚(−)
              − 资源压力(−)
然后 clamp 到 [0, 100]
```

各项系数应取「正常经营下缓慢波动、长期可生存，疏于管理则逐步逼近警报线」的平衡值，并在代码注释中标明含义以便调参。
